Use Cases

Patterns for research, docs, and org memory

Academic Research

Ingest papers, build knowledge graphs, and generate grounded summaries.

from hidden_brain import RAG

rag = RAG.from_papers(["paper1.pdf", "paper2.pdf"])
summary = rag.query("What are the key findings?")

Technical Documentation

Index codebases, answer developer questions, and draft documentation from sources.

from hidden_brain import RAG

rag = RAG.from_codebase("./src")
answer = rag.query("How does authentication work?")

Knowledge Management

Build organizational memory, connect decisions, and retrieve precise context.

from hidden_brain import RAG

rag = RAG.from_directory("./knowledge_base")
context = rag.search("project decisions 2026", limit=5)

Quick Start

Install core packages and run locally

hidden-brain (RAG 4.0)

pip install hidden-brain

docker compose up -d
