The 4 Layers

Security by architecture, not by promises

1

Xokito — Privacy PRE-LLM

What: Detects and anonymizes sensitive data (emails, API keys, git authors) BEFORE the LLM sees anything. Reversible post-LLM for output.

How: Pattern matching + reversible pseudonymization. Graceful degradation (non-blocking).

Why: Privacy by design, not post-hoc filtering. Your secrets never reach the model.

Status: Integration in progress
2

Ethilia — Ethics Validation

What: Risk scoring PRE-execution. Auto-block high-risk requests. Configurable thresholds per organization.

How: Configurable thresholds. Double-check (input + output validation). Policy engine.

Why: Embed ethics in architecture, not as optional plugin. Compliance-ready from day one.

Status: Partial (pipeline mode implemented)
3

GAIA — Coherence & Epistemic Humility

What: Lagrangian coherence (L = T - V) detects narrative contradictions. Shannon entropy classifies certainty (fact/belief/guess).

How: Physics-inspired formulas for measuring narrative consistency. Embedding-based contradiction detection.

Why: Explicit "I know" vs "I don't know" transparency. Auditable uncertainty.

Status: Implemented (integration partial)
4

Cutufato — Audit Trail

What: SHA-256 hashes for every prompt/response. Immutable trail for compliance.

How: Non-blocking logging. Compliance-ready (SOC2, GDPR, HIPAA). Local or remote storage.

Why: Accountability without performance penalty. Full traceability for regulators.

Status: Audit logger implemented (metadata incomplete)
